## version 0.1.1  增加 style 操作(2016-8-2)

## version 0.1.0  开启 Perfect DOM 之旅(2016-6-18)

* 搭建项目(包括打包、发布等)
* 实现以下通用 api
  * checkBodyScrollbar  判断 body 滚动条是否显示
  * scrollbarWidth 获取浏览器滚动条宽度
  * zIndex  获取当前元素 zIndex
  * offset  获取当前元素 offset
* 完善文档
